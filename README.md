CRUD de Empleados en Java con PostgreSQL

Este proyecto consiste en una aplicación desarrollada en Java que implementa las operaciones básicas del CRUD (Crear, Leer, Actualizar y Eliminar) conectada a una base de datos PostgreSQL.
La aplicación cuenta con una interfaz gráfica (GUI) que permite al usuario interactuar fácilmente con la base de datos.

Tecnologías utilizadas:
- Java
- PostgreSQL
- JDBC
- Swing

La aplicación permite:

- Insertar empleados
- Buscar empleados por ID
- Actualizar datos
- Eliminar registros
